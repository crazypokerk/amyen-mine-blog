## 网页设计的重要性
![5-7-2.png](https://cdn.nlark.com/yuque/0/2022/png/25431352/1651888674444-a2a2f293-6cf7-4d09-b1c2-c6b1ca1ec1f1.png#clientId=uca5b391c-bc6f-4&crop=0&crop=0&crop=1&crop=1&from=ui&id=u393058c8&margin=%5Bobject%20Object%5D&name=5-7-2.png&originHeight=728&originWidth=1363&originalType=binary&ratio=1&rotation=0&showTitle=false&size=284924&status=done&style=none&taskId=u27576fd3-b8e8-4c0f-91fe-4cfd6f55cc2&title=)<br />好的设计：

- 可以直接为产品或品牌留下一个长久的好印象；
- 让用户立刻有更多的信任感（确实如此，apple官网就让人有购买的冲动~）；
- 增加用户对品牌的顾客认知价值；
- 给予用户明确的网站意图，如购买产品或搜寻信息等。

坏的设计：

- 使用户不相信产品或服务的可靠性；
- 使用户缺少对产品品牌的信任感；
- 让用户觉得产品有“廉价感”；
- 使用户感到迷惑，无法快速获取有用且有价值的信息。
